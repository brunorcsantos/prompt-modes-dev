# prompt-modes-dev

 🧠 Prompt Modes for AI

Este repositório contém um conjunto de prompts estruturados para Inteligência Artificial, organizados por modos de uso. O objetivo é fornecer instruções claras e reutilizáveis para melhorar a qualidade das respostas da IA conforme o tipo de tarefa desejada.

Os prompts foram desenvolvidos utilizando boas práticas de Prompt Engineering, permitindo interações mais objetivas, previsíveis e produtivas.

🔹 Modos Disponíveis

🟢 ASK

Prompts voltados para:

Perguntas diretas

Explicações técnicas

Diagnóstico de erros

Esclarecimento de dúvidas

Objetivo: obter respostas claras e objetivas da IA.

🔵 PLAN

Prompts voltados para:

Planejamento de projetos

Definição de estratégias

Estruturação de etapas

Organização de tarefas

Objetivo: auxiliar no raciocínio estratégico e no planejamento lógico.

🟡 STUDY

Prompts voltados para:

Aprendizado guiado

Explicações didáticas

Estudos progressivos

Fixação de conteúdo

Objetivo: transformar a IA em uma ferramenta de apoio educacional.

🟣 PLANNER

Prompts voltados para:

Organização pessoal

Definição de metas

Criação de rotinas

Gestão de tempo

Objetivo: apoiar a produtividade e a organização do usuário.

🎯 Objetivo do Projeto

Este projeto foi desenvolvido como parte de uma atividade da plataforma Digital Innovation One (DIO), com a finalidade de:

Demonstrar o uso de repositórios Git.

Utilizar arquivos Markdown para documentação.

Aplicar conceitos de engenharia de prompts.

Criar um material reutilizável para interações com IA.

Organizar prompts por categorias funcionais.

🛠️ Tecnologias Utilizadas

Git e GitHub

Markdown (.md)

Conceitos de Prompt Engineering

Inteligência Artificial (LLMs)

🚀 Como Utilizar

Acesse a pasta prompts.

Escolha o modo desejado (ask, plan, study ou planner).

Copie o conteúdo do arquivo .md.

Cole o prompt na IA de sua preferência (ChatGPT, Copilot, etc).

Adapte conforme sua necessidade.
